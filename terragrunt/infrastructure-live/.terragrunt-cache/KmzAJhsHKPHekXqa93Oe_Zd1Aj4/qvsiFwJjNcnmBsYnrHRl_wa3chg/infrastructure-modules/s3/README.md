# AWS S3

This terraform module helps user to:

- Create S3 bucket
- Setup policy on S3 bucket
- Enable ACL on bucket
- Enable logging in bucket
- Enable versioning in bucket
- Block public access on bucket

## Usage

```hcl
service            = path_relative_to_include()
attributes         = "general"
kms_master_key_arn = <kms-key-arn>
kms_master_key_id  = <kms-key-id>
policy             = <<POLICY
{
    "Version": "2012-10-17",
    "Statement": [
        {
          "Effect": "Deny",
          "Principal": "*",
          "Action": "*",
          "Resource": [
              "arn:aws:s3:::<bucket-name>/*",
              "arn:aws:s3:::<bucket-name>"
          ],
          "Condition": {
              "Bool": {
                  "aws:SecureTransport": "false"
              }
          }
        }
    ]
}
  POLICY
}
```
